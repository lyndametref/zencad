Curve Analysis

Theoretical summary.

The generally accepted method for defining curves in computational geometry systems is the parametric method.

According to it, the curve is given by a continuous mapping of the scalar set [U_min, U_max] onto a space of a given dimension. P = F(U) : P ∈ R^N, U ∈ R^1[U_min, U_max], with F - a mapping function, аnd N - the dimensionality of space.

In practice, this means that any point P on the curve has a corresponding value of the scalar parameter U . It should be understood that, in general, the function connecting the parameter U at point P and the length of the curve from the point of origin O to point P is not linear. ... Therefore, calculations over a curve in terms of lengths require the use of a special mathematical apparatus (implemented in the form of methods in this library).

Curve classes.

ZenCad has the following classes that implement curve analysis methods:

  • Edge (generated by tools segment, interpolate, bezier, bspline etc)
  • Curve3 (NotReleased)
  • Curve2 (NotReleased)

End points and range of a curve.

Defining the endpoints of curves.

The endpoints method returns endpoints objects. The parameters of these points can be queried with the range method.

curve.endpoints() -> point3, point3
curve.range() -> float, float

crv = circle(r=5, wire=True, angle=deg(270))
s,f = crv.endpoints()
disp([crv, s, f])

curve.length()

Return the length of a curve between parameters U_min and U_max.

curve.d0(u)

Return point matching parameter u.

curve.d1(u)

Return the point and vector of the first derivative corresponding to the parameter u.

curve.linoff(u, dist)

Return the coordinates of the point offset by the length dist relative to the point specified by the parameteru.

curve.linoff_point(u, dist)

Rotate a point shifted by the length dist relative to the point specified by the parameter u.
alternate: curve.d0(curve.linoff(u,dist))

Equally distributed points of the curve.

Return an array of points equally spaced along the curve. Parameter npnts - sets the number of points. The parameters umin, umax set the range on the set of parameters in which the distribution procedure will be carried out.

curve.uniform(npnts, umin=U_min, umax=U_max) -> list(float) # Список параметров
curve.uniform_points(npnts, umin=U_min, umax=U_max) -> list(point3) # Список точек.

crv = circle(r=5, wire=True, angle=deg(270))

params = crv.uniform(8, math.pi/4, math.pi)
print(params) # [0.7853981633974483, 1.121997376282069, 1.4585965891666897, 1.7951958020513104, 2.131795014935931, 2.4683942278205517, 2.8049934407051724, 3.141592653589793]

pnts = crv.uniform_points(8, math.pi/4, math.pi)
disp(pnts + [crv])
