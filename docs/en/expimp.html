Export/Import

Export and import operations differ from most operations in the zencad library in that they are performed not only on objects in program memory, but also on files. The evalcache caching library uses a special mechanism to work with files. It allows imports to take into account changes to the content of files, and exports are not performed unnecessarily.

STL

Create an STL mesh file located at path from solid model model . The delta parameter determines the level of detail. The smaller the delta , the smaller the size of the polygons.

to_stl(model, path, delta)

BREP

Create a brep file of BREP format located at path from solid model model . 

to_brep(model, path)

Read the brep file of the BREP format located at the path . Return the resulting model. 

m = from_brep(path)
