Установка

ZenCad system tested on Linux (of the Debian family), Windows.

Installation with pypi

python3 -m pip install zencad

Method of installation on various OS.

Installation on Windows.

At the moment, you may need to install vcredist2019.

Installation on MacOS.

MacOS version in development.

Installing standalone version (Windows).

The

StandAlone version can be found at here . StandAlone version may lag behind the upstream.

Install from source.

To install from source, you need to build or get compiled binary dependencies:
* pyservoce
* OpenCascade(required version)
Compilation features for different OS may differ.
