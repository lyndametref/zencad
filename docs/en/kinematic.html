Kinematics library.

This library is an addition to the assembly system and allows you to control the movement of assembly elements in terms of generalized kinematic coordinates, as well as perform some actions on kinematic chains.

The library defines additional unit types:

zencad.libs.kinematic.rotator - rotary link zencad.libs.kinematic.actuator - actuator link
