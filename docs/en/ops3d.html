Offset.

By offsetting the shells, builds the thickened thinned body based on the prototype proto and the radius of the thickened r . If r is positive, the envelope is shifted outward, if negative - inward.

Signature: 

     offset 
                     (  proto  ,   r  )

Example: 

    offset   (  cone   (  r1   =   15  ,   r2   =   10  ,   h   =   20  ),   r   =   5  )

Ruled surface.

< p> Draws a facet washed ruled surface based on linesa, b.

Signature:

ruled(a, b)

Example:

ruled(circle(r=20, wire=True), circle(r=20, wire=True).up(20))
ruled(circle(r=20, wire=True), circle(r=20, wire=True).rotZ(math.pi/2*3).up(20))
ruled(
    interpolate([(0,0),(-4,10),(4,20),(-6,30),(6,40)]),
    interpolate([(0,0),(-2,10),(2,20),(-4,30),(4,40)]).up(20),
)


Anchoring

Creates a shell on an array of wireframe profiles . When you activate the shell option, a shell is built instead of a solid body. Enabling the smooth option changes the ruled surface mode to a smooth approximation. When approximation is active, the maxdegree option limits the maximum degree of the polynomial.

Signature: 

loft(profiles, smooth=False, shell=False, maxdegree=4)
