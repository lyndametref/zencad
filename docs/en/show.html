Mapping

Scene

Scene - this is a container that stores models and the colors assigned to them. 

scene = Scene()
scene.add(model)
scene.add(model, color)

scene.add returns an interactive object, with which you can change the state of the displayed object (see.interactiveobject)

zencad.default_scene - the scene object used by display related operations, unless otherwise noted.

zencad.display

A tool for adding geometry to a scene.

zencad.display(shape, color=zencad.default_color, scene=zencad.default_scene)
zencad.display(unit, color=zencad.default_color, scene=zencad.default_scene, deep=True)
zencad.display(lst, color=zencad.default_color, scene=zencad.default_scene)

zencad.disp(model) # alternate

Depending on the parameter type, display can behave as follows: For shape - adds to the scene, returns interactive object For assemble.unit - calls bind to_scene, returns None For listtuple - calls display iteratively for list items , returns a list of results.

zencad.highlight

A special variant of the display function used for debugging constructed geometry.

zencad.highlight(model, color=zencad.color(0.5, 0, 0, 0.5))
zencad.hl(model) # alternate naming

Unlike display, the function returns the object passed to it. Usage example (highlight the subtracted form):

c = a - hl(b.up(100500))

color

An object containing color information in rgba format. Parameter value range [0,1].

color(r,g,b,a)

ZenCad defines a standard set of colors:

zencad.color.white =     zencad.color(1,1,1)
zencad.color.black =     zencad.color(0,0,0)
zencad.color.red =       zencad.color(1,0,0)
zencad.color.green =     zencad.color(0,1,0)
zencad.color.blue =      zencad.color(0,0,1)
zencad.color.yellow =    zencad.color(1,1,0)
zencad.color.magenta =   zencad.color(0,1,1)
zencad.color.cian =      zencad.color(1,0,1)
zencad.color.mech =      zencad.color(0.6, 0.6, 0.8)
zencad.color.transmech = zencad.color(0.6, 0.6, 0.8, 0.8)

show

zencad.show(scene=zencad.default_scene,
            animate=None, preanimate=None, close_handle=None)

The show function initiates the display of the renderer program. If the function is called with no arguments, the default scene is displayed (zencad.default_scene).

show also has a set of parameters to support model animation functions (see. Animation).

The behavior of the show function is covered in more detail in ( Inner Kitchen)
